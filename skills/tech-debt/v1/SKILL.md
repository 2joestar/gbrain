# Tech Debt Analyzer

**Trigger:** `/tech-debt <file_path or code>`
**Version:** v1
**Status:** active
**Source:** @KSimback /tech-debt-skill

## Purpose
Thorough code review: finds bugs, code smells, improvements, security issues, performance problems, and maintainability concerns. Returns structured report with severity ratings.

## Usage
```
/tech-debt apple/core/response_orchestrator.py
/tech-debt <paste code here>
```
