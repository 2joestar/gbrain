# Heretic — Uncensored Mode Controller

**Trigger:** `/heretic on|off|status` (admin only)
**Version:** v1
**Status:** active
**Source:** github.com/p-e-w/heretic (20K stars)

## Purpose
Toggle uncensored AI mode. When enabled, suppresses standard AI refusal patterns in system prompts. Switchable — default OFF. Admin-only for safety.

## Usage
```
/heretic on   → Enable uncensored mode
/heretic off  → Disable, return to standard
/heretic      → Show current status
```
